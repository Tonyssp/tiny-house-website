## 🚀 How to Run (Choose One)

### ⭐ Option 1: EASIEST - Double-Click Batch File
1. Find **`run-server.bat`** in your `tiny` folder
2. **Double-click it** to start the server
3. Your browser will automatically open or go to: **http://localhost:8000**
4. Press **CTRL+C** in the command window to stop

### Option 2: Python (Built-in on Windows)
```cmd
cd C:\Users\Tony\Pictures\tiny
python -m http.server 8000
```
Then open: **http://localhost:8000**

### Option 3: Node.js (Optional)
1. Install Node.js from https://nodejs.org/
2. Double-click **`run-server-node.bat`**
   
Or manually:
```cmd
cd C:\Users\Tony\Pictures\tiny
npx http-server -p 8000
```

### Option 4: VS Code Live Server (Development)
1. Open folder in VS Code
2. Install "Live Server" extension
3. Right-click `index.html` → "Open with Live Server"

### Option 5: Direct Open (Limited)
Simply double-click `index.html` in Windows Explorer
(Note: Some features may not work properly this way)

## ⚙️ Troubleshooting

### "Page looks blank"
- Make sure you're using a **local server** (not opening directly)
- Try reloading the page (Ctrl+R)

### "Images not showing"
- Verify `/images/` folder has 93 files
- Check browser console for 404 errors (F12)
- Ensure server is running at http://localhost:8000

### "Lao text not displaying correctly"
- Browser should auto-load Google Fonts
- Clear browser cache (Ctrl+Shift+Delete)
- Try different browser (Chrome, Firefox, Edge)

### "Server won't start"
- Check if port 8000 is already in use
- Try: `python -m http.server 9000` (different port)
